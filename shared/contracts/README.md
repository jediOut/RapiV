# @rapidin/contracts

Contratos compartidos entre backend, cliente, negocio y repartidor.

Usa este paquete para tipos que cruzan la API:

- usuarios, roles y sesion
- negocios y productos
- pedidos, subpedidos, estados y payloads
- constantes de estado que usan varios proyectos

No agregues aqui componentes, estilos, hooks, navegacion ni funciones HTTP. Eso pertenece a cada app.
